## Built With

- HTML/CSS
- JavaScript
- Adobe XD for design

- Working on recreating the website to ReactJS/TypeScript

### Installing

1. Clone the repo:

```bash
git clone https://github.com/Benjamel/Portfolio.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
